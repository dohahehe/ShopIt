## üìã Overview

FreshCart is a full-featured e-commerce platform built with modern web technologies. It provides a seamless shopping experience with user authentication, product browsing, cart management, wishlist functionality, and secure checkout processing.

## ‚ú® Key Features

### Shopping Experience
- Browse products with filtering and search
- Product categories and brands navigation
- Detailed product pages with image galleries
- Add to cart and wishlist functionality
- Real-time cart updates with quantity management

### User Management
- Secure authentication with NextAuth.js
- User registration and login
- Profile management with editable information
- Password change functionality
- Order history tracking

### Order Processing
- Checkout with shipping address management
- Multiple payment methods support
- Order confirmation

### Reviews & Ratings
- User reviews with star ratings
- Edit and delete own reviews
- Admin moderation capabilities
- Average rating display on products

### Wishlist
- Save favorite products
- Add/remove items from wishlist
- Quick add to cart from wishlist
- Persistent across sessions
